# Test Reference Files

Reference files were generated by running `pprc` (index.js) from the `tests/in/` directory.

Run `npm test` to verify output matches references (byte-for-byte).

## Regenerating references

All commands run from `tests/in/`:

```bash
# default (shared profiling via negpro)
node ../../index.js --output-dir ../reference/default

# per-image balancing (negpro with per-image profiling)
node ../../index.js --per-image-balancing --output-dir ../reference/per-image-balancing

# no-invert (raw tiff, no negpro)
node ../../index.js --no-invert --output-dir ../reference/no-invert

# gamma1 (no gamma correction, no negpro)
node ../../index.js --gamma1 --no-invert --output-dir ../reference/gamma1

# e6 (auto-level, no negpro)
node ../../index.js --e6 --output-dir ../reference/e6

# bw (invert, auto-level, greyscale, no negpro)
node ../../index.js --bw --output-dir ../reference/bw

# bw-rgb (invert, auto-level, RGB, no negpro)
node ../../index.js --bw-rgb --output-dir ../reference/bw-rgb
```
